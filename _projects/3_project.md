This project presents the first large-scale analysis of substance-related hashtag communities on TikTok. Using social network analysis across 2,333 hashtags and 39,509 videos, we identified 16 distinct hashtag communities — ranging from harm reduction advocacy to recreational use content — and characterized their structural and thematic relationships.

Available as a preprint on **arXiv** {% cite bouzoubaa2025dr00gtiktok %}.

**Methods:** Social network analysis, qualitative content coding, large-scale data collection.

**Collaborators:** Layla Bouzoubaa, Muqi Guo, Afsaneh Razi, Rezvaneh Rezapour
