Studying social media behavior at scale is hampered by platform API restrictions and inconsistent data access. This project develops a platform-agnostic data donation tool that enables researchers to collect consented, user-provided social media data — regardless of platform — opening new methodological possibilities for HCI and social computing research.

**Advisor:** Dr. Afsaneh Razi (DONUTS Lab, Drexel University)
